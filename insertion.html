🃏 插入排序法

Insertion Sort — 視覺化步驟展示

← 回首頁

原理：把資料分成已排序區與未排序區，每次取出一個 key，向左比較並插入到正確位置。

時間複雜度：平均 / 最差為 O(n²)，最佳可達 O(n)。

for i = 1 to n-1
  key = arr[i]
  j = i - 1
  while j >= 0 and arr[j] > key
    arr[j+1] = arr[j]
    j--
  arr[j+1] = key
