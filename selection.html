原理： 選擇排序法每一輪會從尚未排序的部分找出最小值， 再與未排序區域的第一個元素交換位置。每輪結束後，最小值被擺到正確位置， 已排序區域就擴大一格。

時間複雜度： 最差 / 平均 / 最佳均為 O(n²)。 每輪雖然比較很多次，但通常最多 1 次交換。

function selectionSort(arr) {
  let n = arr.length;

  for (let i = 0; i < n - 1; i++) {
    let minIdx = i;

    for (let j = i + 1; j < n; j++) {
      if (arr[j] < arr[minIdx]) {
        minIdx = j;
      }
    }

    if (minIdx !== i) {
      [arr[i], arr[minIdx]] = [arr[minIdx], arr[i]];
    }
  }

  return arr;
}
